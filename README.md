# JSON/XML 在线格式化工具

在线格式化和验证 JSON、XML、SQL 的工具。

网址：http://json.ofcoder.com/

## 新功能：对象/数组折叠

新增了对 JSON 和 XML 对象/数组的展开和折叠功能！

### 功能特点

- **一键折叠/展开**：点击行号前的 `−` 或 `+` 按钮可以折叠或展开单个对象/数组
- **批量操作**：使用右侧的 `[ + ]` 和 `[ − ]` 按钮可以一次性展开或折叠所有内容
- **智能识别**：自动识别所有可折叠的对象 `{}` 和数组 `[]`
- **数据保护**：折叠仅改变显示方式，不会修改原始数据

### 使用方法

1. 在文本框中输入或粘贴 JSON/XML 内容
2. 点击 `Lint Me!` 按钮进行格式化
3. 格式化成功后即可使用折叠功能：
   - 点击行前的 `−` 按钮折叠单个区域
   - 点击右侧的 `[ − ]` 按钮折叠所有内容
   - 点击右侧的 `[ + ]` 按钮展开所有内容

查看 [详细功能说明](FOLD_FEATURE.md) 了解更多信息。
